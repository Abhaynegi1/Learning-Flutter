# LockIn

A competitive focus timer app built with Flutter.

## Features
- **Solo Mode Focus**: Choose between 15, 25, 45, or 60-minute sessions.
- **Anti-Distraction Detection**: If you leave the app, minimize it, or switch tasks, the session fails instantly.
- **Streak System**: Build your focus streak by completing successful sessions.
- **History**: Track your past wins and losses.
- **Premium UI**: Modern dark theme with smooth animations and transitions.

## Getting Started
To run the project:
1. Ensure you have Flutter installed.
2. Clone the repo and run `flutter pub get`.
3. Connect a device/emulator and run `flutter run`.
